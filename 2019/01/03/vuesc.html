模板解析

我们经常在vue中使用vue模板，比如：

模板的本质是字符串，其特点有：

  • 包含 if/for 等逻辑指令
  • 动态的（动态生成 dom ）
  • 最终转换为 html

综上，要将模板转换为 html ，只能使用 js 来操作。在 html/css/js 中，只有js能处理逻辑和渲染页面。
